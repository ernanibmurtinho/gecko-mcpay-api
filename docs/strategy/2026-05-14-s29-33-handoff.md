## The addendum text (verbatim)

```
QUANTITATIVE GROUNDING (mandatory for every non-coordinator voice):
  When you write a specific number — TVL, APR/APY, price, volume,
  ratio, percentage, dollar figure — you MUST either:
    (a) QUOTE the exact substring from the cited chunk that contains
        that number, in the same line — e.g.
        `TVL is $1,505,498,759 [3: 'tvl: 1505498759']`, OR
    (b) Write the claim WITHOUT a citation and prefix it `unsourced:` —
        e.g. `unsourced: yields appear competitive vs peers`.
  Do NOT cite a chunk number for a figure that does not appear verbatim
  in that chunk. Inferring a plausible TVL/APR/price from field names,
  related magnitudes, or surrounding context is a hallucination and
  disqualifies the turn. When in doubt, write the qualitative claim
  with `unsourced:` rather than fabricate a specific number.
```

Inserted into `_opening_prompt` in
`packages/gecko-core/src/gecko_core/orchestration/trade_panel/__init__.py`
immediately after the existing CITATION DISCIPLINE block. Same structural
shape (panel-wide directive, every persona sees it once), so no persona-
prompt JSON file was touched.
